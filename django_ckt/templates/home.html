Key Features

  • Interactive Visualization: Zoom, pan, and explore causal graphs
  • Graph Editing: Remove nodes and edges, undo changes
  • Causal Analysis: Calculate adjustment sets and instrumental variables
  • SemMedDB Integration: Generate graphs from biomedical literature
  • Export Options: Save as R files or HTML reports

System Information

Framework: Django 5 with ASGI
R Integration: rpy2 with visNetwork, dagitty, igraph
Database: PostgreSQL (SemMedDB)
Frontend: Bootstrap 5 + vis-network
