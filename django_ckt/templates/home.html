About CKT - Causal Knowledge Trace

What is CKT?

CKT (Causal Knowledge Trace) is an interactive web application for exploring and analyzing causal relationships in knowledge graphs. It provides tools for visualizing graphs (which may contain cycles), performing causal inference analysis, and understanding complex relationships between variables extracted from biomedical literature.

Key Features

  • Interactive Graph Visualization: Explore causal graphs with zoom, pan, and physics-based layouts
  • Graph Editing: Remove nodes and edges, undo changes, and save modified graphs
  • Causal Analysis: Calculate adjustment sets, find instrumental variables, and analyze causal paths
  • Graph Configuration: Generate knowledge graphs from biomedical databases with customizable parameters
  • Multiple Export Options: Save graphs as R files or HTML reports

User Guide

Step 1: Graph Configuration (Optional)

If you want to generate a new knowledge graph from the SemMedDB database:

  1. Navigate to the Graph Configuration tab
  2. Enter the Exposure CUI (Concept Unique Identifier) - you can search for a word and select from available CUIs
  3. Enter the Outcome CUI - you can search for a word and select from available CUIs
  4. Configure optional parameters:
    • Squelch Threshold: Minimum number of unique PMIDs (publications) required for an edge
    • Publication Year Cutoff: Only include publications from this year onwards
    • Degree: Maximum distance from exposure/outcome nodes to include
    • SemMedDB Version: Select the database version to use
  5. Click Generate Graph to create your knowledge graph
  6. Wait for the process to complete - this may take several minutes

Note: If you already have a graph file, you can skip this step and go directly to Data Upload.

Step 2: Data Upload

Load a graph file into the application:

  1. Navigate to the Data Upload tab
  2. Choose one of two methods:
    • Method 1 - Select Existing File: Choose a file from the dropdown (files in graph_creation/result directory) and click 'Load Selected Graph'
    • Method 2 - Upload New File: Use the file upload interface to upload an R file containing your DAG definition
  3. Optional: Apply filtering to remove leaf nodes (nodes with only one connection)
  4. Wait for the graph to load - you'll see a progress indicator

Required File Format: Your R file must contain a dagitty graph definition assigned to variable 'g'

Step 3: Graph Visualization

Explore and modify your causal graph:

  1. Navigate to the Graph Visualization tab
  2. Interact with the graph:
    • Zoom: Use mouse wheel or pinch gesture
    • Pan: Click and drag the background
    • Select Nodes/Edges: Click on them to view details
    • Move Nodes: Drag nodes to reposition them
  3. View edge information in the table below the graph
  4. Modify the graph:
    • Select a node and click Remove Selected Node
    • Select an edge and click Remove Selected Edge
    • Click Undo Last Removal to revert changes
  5. Adjust physics settings for better layout
  6. Save your work:
    • Save DAG: Download modified graph as an R file
    • Save HTML: Export as a readable HTML report

Node Colors: Red = Exposure | Cyan = Outcome | Gray = Other variables

Step 4: Causal Analysis

Perform statistical causal inference analysis:

  1. Navigate to the Causal Analysis tab
  2. Select your Exposure Variable from the dropdown
  3. Select your Outcome Variable from the dropdown
  4. Choose the Effect Type (Total Effect or Direct Effect)
  5. Run analyses:
    • Calculate Adjustment Sets: Find variables to control for to estimate causal effects
    • Find Instrumental Variables: Identify variables for instrumental variable analysis
    • Analyze Causal Paths: Examine all paths between exposure and outcome
    • Run Complete Analysis: Execute all analyses at once
  6. Review results in the tabbed panels

Understanding Causal Concepts

Adjustment Sets

What it is: A set of variables to control for in your analysis. When you control for these variables (e.g., by including them in a regression model), you can estimate the causal effect of the exposure on the outcome, blocking all backdoor paths that create spurious associations.

Instrumental Variables

What it is: A variable that affects the outcome only through its effect on the exposure, and is not associated with any confounders. IVs can be used to estimate causal effects when randomization is not possible and confounding is present.

Causal Paths

What it is: All directed paths from the exposure to the outcome. These paths represent the mechanisms through which the exposure may causally affect the outcome. Understanding these paths helps identify mediators and direct vs. indirect effects.

Useful Links

Example: Visit the dagitty.net website - a popular tool for drawing and analyzing causal diagrams. Many concepts in CKT are based on dagitty's methodology.
