Instructions

Method 1: Place files in graph_creation/result directory

  • Create R files (e.g., Aspirin_1.R, Aspirin_2.R, Vegans_1.R, Manualbibset_links.R) with your DAG definition
  • Place in the graph_creation/result directory (generated graphs are automatically saved here)
  • Click 'Refresh File List' and select your file
  • Click 'Load Selected DAG'

Method 2: Upload files through the interface

  • Use the file upload interface above
  • Select your R file containing the DAG
  • Click 'Upload & Load'
DAG File Format

Your R file should contain a dagitty graph definition like:

g <- dagitty( "dag { Variable1 [exposure] Variable2 [outcome] Variable3 Variable4 Variable1 -> Variable2 Variable2 -> Variable3 Variable3 -> Variable4 }" )

The variable name must be 'g' for the app to recognize it.
